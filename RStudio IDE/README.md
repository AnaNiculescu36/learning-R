# RStudio IDE 

## Important 

- restart your session often with Session / Restart R or Ctrl-Shift-F10
- Useful shortcuts:
    + Ctrl-I to re-indent the line
    + Ctrl-/ to comment the line (custom, re-mapped!)
    + review all of the shortcuts with Alt-Shift-K
